# 🔗 Dev Links

Projeto de pagina de links pessoais desenvolvido com HTML, CSS e JavaScript.

O objetivo do projeto e reunir links importantes em uma interface simples, responsiva e com alternancia entre tema claro e escuro.

## 🚀 Tecnologias

- HTML
- CSS
- JavaScript
- Ionicons
- Git e GitHub

## ✨ Funcionalidades

- 👤 Exibicao de perfil com avatar
- 🔗 Lista de links principais
- 🌐 Links para redes sociais
- 🌓 Alternancia entre modo claro e escuro
- 🖼️ Imagens diferentes para cada tema

## 📁 Estrutura do Projeto

```text
projeto-discover/
├── assets/
├── css/
├── html/
├── js/
└── README.md
```

- `assets/`: imagens, fundos e icones
- `css/`: arquivo de estilos da pagina
- `html/`: arquivo HTML principal
- `js/`: arquivo JavaScript do projeto

## 🧭 Como Abrir

Abra o arquivo `html/index.html` no navegador.
